# **App Name**: StayFloow

## Core Features:

- Intuitive Multi-Category Search & Filtering: A unified search bar with intelligent auto-suggest, dynamic filters, and integrated mapping across accommodations, car rentals, and circuits, enabling users to easily find listings.
- Dynamic Partner Listing Form (Smart Onboarding): A guided, mobile-friendly onboarding experience for partners to list accommodations, cars, or circuits via a single smart form with category-specific fields, address autocomplete with GPS (OpenStreetMap/Nominatim), and real-time availability calendar integration.
- Multi-Photo Upload & Management: Allows partners to upload multiple listing photos (minimum 5, up to 30) via drag & drop with preview, reordering, and hero image selection. Photos are optimized for various devices and low-bandwidth networks.
- AI-Powered Description & Promotion Tools: Provides an AI tool to assist partners in enhancing their listing descriptions for clarity and appeal, and suggests tailored promotional messages (e.g., 'Flash Sale for Eid') to boost visibility.
- Admin Listing Validation Dashboard: A secure administrative interface for manual review and approval of pending partner listings, including the ability to reject with feedback or request modifications, ensuring quality and accuracy.
- Comprehensive Listing Detail Pages: Dedicated pages for each listing type featuring rich photo galleries, detailed descriptions, amenities displayed as icons, house rules, reviews, and a clear, interactive availability calendar reflecting partner-defined periods.
- Frictionless Booking & Secure Payment Flow: A multi-step booking process with a progress bar, guest detail autofill, transparent pricing, booking rules summary, and secure integration with local payment gateways (e.g., Stripe, Fawry) for cards and mobile money.

## Style Guidelines:

- Primary interactive color: A deep, authoritative blue (#003580) for calls-to-action, active elements, and branding, embodying trust and reliability, inspired by Booking.com's aesthetic. HSL(210, 100%, 25%).
- Background color: A very subtle, almost off-white bluish-grey (#F9F9FA) to provide a clean canvas that makes 'cartes blanches' (white cards) stand out, contributing to a modern and light feel. HSL(210, 5%, 98%).
- Accent color: A vibrant turquoise (#4EEAEA) to provide visual interest and contrast for secondary highlights, subtle animations, or gamified elements. HSL(180, 80%, 55%).
- Body and headline font: 'Inter' (sans-serif) for its modern, objective, and highly legible characteristics, suitable for detailed information and a clean user interface across all text elements.
- Utilize a consistent set of clean, modern, flat vector icons for amenities and features, similar to Booking.com, to enhance usability and information at a glance. Subtle African-inspired patterns may be integrated into non-interactive UI backgrounds rather than the icons themselves for a sophisticated touch.
- Adopt a mobile-first, card-based grid layout with clear content segmentation, prominent photo sliders, interactive maps (OpenStreetMap), and intuitive navigation through multi-step forms via progress bars. Emphasize responsive design across all devices.
- Incorporate subtle, tasteful UI animations to improve user engagement and provide visual feedback. Examples include smooth transitions between search results, interactive gamification elements (e.g., 'heart' icons for favorites), and delightful micro-interactions during the booking process.
